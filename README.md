# M06 Programming Assignment – B-Tree Insert

## Description
- **Goal:** Implement a B-Tree data structure with a working insert operation.
- **Language:** C++
- **Key features:** Insert, search, and traversal of a B-Tree of integers.

## Inputs and Outputs
- **Input:** Hard-coded integer sequences in `main.cpp` (can be extended to user input).
- **Output:** 
  - B-Tree traversal printed to the console after insert sequences.
  - Search result for a specific key.

## How to Compile and Run

### Using g++
```bash
cd src
g++ -std=c++17 main.cpp BTree.cpp -o btree_app
./btree_app
